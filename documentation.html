How to use?

Download all required files from here.
HTML   
<div data-hybicon="github-star"></div>

How to customize?

Icon

You can use predefined hybrid icons with data-hybicon attribute. 
<div data-hybicon="github-star"></div>
See available icons

Hover mode

The secondary icon animated on hover when set data-hybicon-hovermode attribute. 
<div data-hybicon="github-star" data-hybicon-hovermode="show|rotate|switch"></div>

Click mode

The secondary icon animated on click when set data-hybicon-clickmode attribute. 
<div data-hybicon="github-star" data-hybicon-clickmode="show|switch"></div>

Info mode

The info field appears when set data-hybicon-infomode attribute. 
<div data-hybicon="github-star" data-hybicon-infomode="show|right-*width*"></div>

Hyperlink

Simple add an anchor tag around hybicon's div. 
<a href="https://github.com/softwaretailoring/hybicon">
    <div data-hybicon="github-star"></div>
</a>

Size

Here are some predefined sizes in hybicon classes. 
<div data-hybicon data-hybicon-size="unit in px|'R' as responsive"" ></div>

Alignment

<div data-hybicon data-hybicon-align="left|center|right"" ></div>

Color

<div data-hybicon="github-star" data-hybicon-background="#fff" data-hybicon-color="#4183c4"></div>

Border

<div data-hybicon="github-star" data-hybicon-border="solid 3px #333333"></div>

Border's radius

<div data-hybicon="github-star" data-hybicon-borderradius="15%"></div>

CSS

#githubstar > svg > [id$=icon1],
#githubstar > svg > [id$=icon2] { 
    fill: #777;
    stroke: #FFF;
}
