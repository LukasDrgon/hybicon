How to use?

Download all required files from here.
Head   
<head>
    <script src="https://cdn.jsdelivr.net/raphael/2.1.4/raphael.min.js"></script>
    <script src="https://cdn.jsdelivr.net/hybicon/1.0.0/hybicon.min.js"></script>
</head>
Body   
<body>
    <div data-hybicon="github-star"></div>
</body>

How to customize?

Icon

You can use predefined hybrid icons with data-hybicon attribute. 
<div data-hybicon="icon1-icon2"></div>
github
star
github-star
See available icons

Hover mode

The secondary icon animated on hover when set data-hybicon-hovermode attribute. 
<div data-hybicon="code-fave" data-hybicon-hovermode="show|rotate|switch"></div>
default
rotate
switch

Click mode

The secondary icon animated on click when set data-hybicon-clickmode attribute. 
<div data-hybicon="github-star" data-hybicon-clickmode="show|switch"></div>
default
switch

Info mode

The info field appears when set data-hybicon-infomode attribute. 
<div data-hybicon="download-code" 
                        data-hybicon-infomode="show|right-*width*"
                        data-hybicon-infotext="*your text*">
</div>
default
right
right-300

Hyperlink

Simple add an anchor tag around hybicon's div. 
<a href="https://github.com/softwaretailoring/hybicon" target="_blank">
    <div data-hybicon="github-star"></div>
</a>
click it :)

Alignment

<div data-hybicon data-hybicon-align="left|center|right"" ></div>
left
default
right

Size

<div data-hybicon data-hybicon-size="unit in px|'css' for responsive behaviour"" ></div>
40

80
default

Background

<div data-hybicon="github-star" data-hybicon-background="as CSS background"></div>
#FFF

Color

<div data-hybicon="github-star" data-hybicon-color="as CSS color"></div>
#4183c4

Border

<div data-hybicon="github-star" data-hybicon-border="as CSS border"></div>
solid 3px #333

Border's radius

<div data-hybicon="github-star" data-hybicon-borderradius="as CSS border-radius"></div>
15%

CSS

<div id="hybiconCSS" 
    data-hybicon="user-idea" 
    data-hybicon-hovermode="switch" 
    data-hybicon-infomode 
    data-hybicon-size="css">
</div>
#hybiconCSS > svg {
    background: #FFF;
    border: solid 4px #333;
    border-radius: 15%;
}
#hybiconCSS > svg > [id$=icon1] { 
    fill: #333;
}
#hybiconCSS > svg > [id$=icon2] { 
    fill: #FFF;
    stroke: #F6921E;
    stroke-width: 0.5;
}
#hybiconCSS > svg > [id$=info] { 
    fill: #8c3d2f;
    stroke: #8c3d2f;
    stroke-width: 3;
}
#hybiconCSS > svg > [id$=infotext] > tspan {
    fill: #FFF;
    font: 13px Impact, sans-serif;
}
