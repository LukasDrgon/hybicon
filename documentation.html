How to use?

Head   
<head>
    <script src="https://cdn.jsdelivr.net/raphael/2.1.4/raphael.min.js"></script>
    <script src="https://cdn.jsdelivr.net/hybicon/1.0.0/hybicon.min.js"></script>
</head>
Body   
<body>
    <div data-hybicon="github-star"></div>
</body>

How to customize?

Icon

You can use predefined hybrid icons with data-hybicon attribute. 
<div data-hybicon="icon1-icon2"></div>
github
star
github-star
See available icons

Hover mode

The secondary icon animated on hover when set data-hybicon-hovermode attribute. 
<div data-hybicon="code-fave" data-hybicon-hovermode="show|rotate|switch"></div>
default
rotate
switch

Click mode

The secondary icon animated on click when set data-hybicon-clickmode attribute. 
<div data-hybicon="icon1-icon2" data-hybicon-clickmode="show|switch"></div>
default
switch

Info mode

The info field appears when set data-hybicon-infomode attribute. 
<div data-hybicon="download-code" 
     data-hybicon-infomode="show|right-*width*"
     data-hybicon-infotext="*your text*">
</div>
default
right
right-300

Hyperlink

Simple add an anchor tag around hybicon's div. 
<a href="github.html"><div data-hybicon="githubalt-link"></div></a>

Animation

Hybicon works on a 100*100 matrix. Animation based on a following formula: "centerX, centerY, size, rotate". 
<div data-hybicon="arrowright-arrowleft"
     data-hybicon-icon1init="20,50,25,0" <!-- default: "45,55,77,0" -->
     data-hybicon-icon1anim="35,50,30,180" <!-- default: "45,55,77,0" -->
     data-hybicon-icon2init="80,50,25,0" <!-- default: "80,20,33,0" -->
     data-hybicon-icon2anim="65,50,30,-180" <!-- default: "80,20,33,0" -->
     data-hybicon-clickmode>
</div>
click it

3rd party icon

Icons are predefined SVG paths. You can use your own icons with data-hybicon-iconclass attribute. 
var myIcons = {
    triangle: "M0,100,L100,100,L50,20,z",
    rectangle: "M0,0,L100,0,L100,100,L0,100,z"
}
<div data-hybicon-iconclass="myIcons"
     data-hybicon="triangle-rectangle"
     data-hybicon-clickmode="rotate" >
<div>
click it

CSS

<div id="hybiconCSS"
     data-hybicon="user-idea" 
     data-hybicon-hovermode="switch" 
     data-hybicon-infomode 
     data-hybicon-size="css">
</div>
#hybiconCSS {
    width: 222px;
    height: 222px;
}
#hybiconCSS > svg {
    background: #FFF;
    border-radius: 15%;
}
#hybiconCSS > svg > [id$=icon1] { 
    fill: #333;
}
#hybiconCSS > svg > [id$=icon2] { 
    fill: #FFF;
    stroke: #F6921E;
    stroke-width: 0.5;
}
#hybiconCSS > svg > [id$=info] { 
    fill: #604c3b;
    stroke: #604c3b;
    stroke-width: 3;
}
#hybiconCSS > svg > [id$=infotext] > tspan {
    fill: #FFF;
    font: 11px Impact, sans-serif;
}
