How to use?

Download all required files from here.
HTML   
<div id="githubstar" class="hybicon100" data-hybicon="github-star"></div>
JavaScript   
window.onload = function () {
    hybicons();
};

How to customize?

Icon

You can use predefined hybrid icons with data-hybicon attribute.
HTML   
<div id="githubstar" class="hybicon100" data-hybicon="github-star"></div>
See available icons

Size

Here are some predefined sizes in hybicon classes.
HTML   
<div id="githubstar" class="hybicon100" data-hybicon="github-star"></div>
See available sizes

Alignment

HTML   
<div id="githubstar" class="hybicon100 hybicon-center" data-hybicon="github-star"></div>
See available alignments

Hover mode

The secondary icon appears on hover when set data-hybicon-hovermode attribute.
HTML   
<div id="githubstar" class="hybicon100" data-hybicon="github-star" data-hybicon-hovermode></div>
See available icons in hovermode

Click mode

The secondary icon appears on click when set data-hybicon-clickmode attribute.
HTML   
<div id="githubstar" class="hybicon100" data-hybicon="github-star" data-hybicon-clickmode></div>
See available icons in clickmode

Hyperlink

HTML   
<div id="githubstar" class="hybicon100" data-hybicon="github-star">
    <a href="sizes.html?hybicon=github-star"></a>
</div>

Color

HTML   
<div id="githubstar" class="hybicon100" data-hybicon="github-star" data-hybicon-color="#777"></div>
CSS   
#githubstar > svg > [id$=icon1],
#githubstar > svg > [id$=icon2] { 
    fill: #777;
    stroke: #FFF;
}
